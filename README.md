# cash-flow-forecasting-app
A web app that generates a 13-week rolling cash forecast from accounting data and highlights upcoming liquidity risks.

## Problem
Small businesses struggle to predict cash shortages due to delayed payments and manual spreadsheets.

## Features
- CSV import for invoices, bills, and cash balances
- Configurable payment-delay assumptions
- 13-week cash forecast
- Alerts for projected cash shortfalls
